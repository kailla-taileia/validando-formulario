# 📄 Formulário de Cadastro com Validação Completa (JavaScript)

## 🚀 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de **colocar em prática os estudos sobre Classes em JavaScript**, aplicando conceitos de orientação a objetos em um cenário mais próximo do mundo real.

A proposta foi criar um **formulário de cadastro completo**, com validações robustas e organização de código, saindo da teoria e enfrentando desafios práticos do desenvolvimento.

---

## 🎯 Objetivos

* Praticar o uso de **classes em JavaScript**
* Aplicar validações reais de formulário
* Melhorar a organização e reutilização de código
* Simular um comportamento próximo de aplicações reais

---

## ⚙️ Funcionalidades

*✔️ Validação de campos obrigatórios (não permite envio vazio)
*✔️ Campo **Nome e Sobrenome** aceita apenas letras (com ou sem acento)
*✔️ Campo **Usuário** permite apenas letras e números (sem símbolos)
*✔️ Controle de **tamanho mínimo e máximo** para usuário e senha
*✔️ Validação de **confirmação de senha**
*✔️ Integração de **validador de CPF**
*✔️ Máscara automática no CPF durante a digitação
*✔️ Bloqueio de envio caso o CPF seja inválido
*✔️ Armazenamento dos dados no **LocalStorage** após envio válido

---

## 🧠 Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Uso de **classes e métodos**
* Manipulação do DOM
* Eventos em formulários
* Validações customizadas
* Reutilização de código (validador de CPF)
* Experiência do usuário (feedback visual)

---

## 💡 Desafios enfrentados

Este foi um projeto **desafiador**, principalmente por ainda estar em processo de aprendizado com classes.

Em alguns momentos, a implementação se tornou complexa e até um pouco “trabalhosa”, mas isso foi essencial para:

* Fixar melhor os conceitos
* Entender na prática como estruturar o código
* Desenvolver raciocínio lógico

---

## 🔐 Segurança e dados

⚠️ Este projeto utiliza **dados fictícios** para fins de estudo.

* Nenhuma informação é enviada para servidores
* Todos os dados são armazenados apenas no **LocalStorage do navegador**
* Não utilize dados reais ao testar a aplicação
